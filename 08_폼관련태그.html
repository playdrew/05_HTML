폼 관련 태그

        form 태그는 html 에서 사용자가 값을 입력 할 수 있는 
        양식을 제공하는 태그이다.
        form 태그의 input 태그를 사용해서 사용자가 입력한
        데이터를 Server 로 전달하는 역할을 하게 된다.(사용자가 입력할 것을 컨트롤러로 전달하는 역할)
        action 속성
            - 폼의 입력 된 값들을 전송받을 서버의 클래스명을 입력
        method 속성
            - get/post 방식으로 서버전송 방식을 지정
            - get 방식 : url 에 넘어가는 데이터가 보인다. (a는 get 방식이다 url 에 보이니까요)
            - post 방식 : url 에 넘어가는 데이터가 감춰진다. (아이디와 패스워드가 url에 노출되면 안되겠죠? 민감한 정보는 post 단순검색은 get방식으로 하게 됩니다.)
    
            submit 이라는 버튼을 누르면 action 에 작성한 경로로
            method 지정 방식에 따라 input 태그에 입력한 값을
            서버로 전송을 하게 된다.
